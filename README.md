## project structure:

```
├── cmd
│   └── main.go
├── go.mod
├── README.md
└── scanner
    └── scanner.go
```

## installation

- use go get:

```
go get github.com/kenjitheman/network_scanner
```

- or use git clone:

```
git clone https://github.com/kenjitheman/network_scanner
```

## usage

- use go run:

```
cd cmd
go run main.go
```
